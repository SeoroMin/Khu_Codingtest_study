## 스터디 규칙
#### 이론
- 각자 공부하기
- 블로그에 올리기
- 발제자는 이론 발제 준비

#### 코드
- 미리 풀어서, 깃허브에 올리기
- 블로그에 따로 정리
- 블로그에 댓글로 피드백
- 스터디 시간에는 서로 리뷰만(모르는 것)

#### 문제
- 발제자가 문제 하나 가져오기
- 다같이 풀기 
- 모르는 거 물어보면서 발제자가 해설 진행
